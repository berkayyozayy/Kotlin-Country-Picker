# Country Picker Application
Android Country Picker Application that displays a list of all the countries using the clean MVVM architecture.
Application can receive data from an API with Retrofit.

## Built with
1) MVVM Architecture
2) Room
3) Gson - Retrofit
4) LiveData
5) Data Binding
6) Kotlin Extensions
7) Kotlin Coroutines
